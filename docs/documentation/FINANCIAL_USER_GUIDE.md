# 📊 Οδηγός Χρήστη - Οικονομικό Σύστημα

## 🏗️ Επισκόπηση

Το Οικονομικό Σύστημα είναι μια ολοκληρωμένη λύση για τη διαχείριση των οικονομικών λειτουργιών σε πολυκατοικίες. Το σύστημα παρέχει:

- **Διαχείριση Δαπανών**: Εγγραφή και παρακολούθηση δαπανών
- **Διαχείριση Πληρωμών**: Εγγραφή πληρωμών από διαμερίσματα
- **Μετρήσεις**: Διαχείριση μετρητών θέρμανσης
- **Αυτοματοποιημένοι Υπολογισμοί**: Μερίδια κοινοχρήστων
- **Αναφορές**: Στατιστικά και γραφήματα
- **Dashboard**: Γενική κατάσταση οικονομικών

---

## 🚀 Έναρξη

### Πρόσβαση στο Σύστημα

1. **Σύνδεση**: Εισέλθετε στο σύστημα με τα διαπιστευτήριά σας
2. **Επιλογή Κτιρίου**: Επιλέξτε το κτίριο που θέλετε να διαχειριστείτε
3. **Οικονομικό Τμήμα**: Μεταβείτε στην ενότητα "Οικονομικά"

### Αρχική Οθόνη

Η αρχική οθόνη περιλαμβάνει:
- **Dashboard**: Γενική κατάσταση οικονομικών
- **Γρήγορες Ενέργειες**: Δημιουργία δαπάνης, εγγραφή πληρωμής
- **Πρόσφατες Κινήσεις**: Τελευταίες δαπάνες και πληρωμές

---

## 💰 Διαχείριση Δαπανών

### Δημιουργία Νέας Δαπάνης

1. **Επιλογή**: Κάντε κλικ στο κουμπί "Νέα Δαπάνη"
2. **Συμπλήρωση Φόρμας**:
   - **Τίτλος**: Περιγραφικός τίτλος της δαπάνης
   - **Ποσό**: Το συνολικό ποσό της δαπάνης
   - **Κατηγορία**: Επιλέξτε την κατάλληλη κατηγορία
   - **Τύπος Κατανομής**: Πώς θα κατανεμηθεί η δαπάνη
   - **Ημερομηνία**: Ημερομηνία της δαπάνης
   - **Περιγραφή**: Επιπλέον λεπτομέρειες

3. **Αποθήκευση**: Κάντε κλικ στο "Αποθήκευση"

### Κατηγορίες Δαπανών

- **Ηλεκτρικό**: Λογαριασμοί ηλεκτρικού
- **Νερό**: Λογαριασμοί νερού
- **Θέρμανση**: Λογαριασμοί θέρμανσης
- **Καθαρισμός**: Υπηρεσίες καθαρισμού
- **Συντήρηση**: Έργα συντήρησης
- **Ασφάλεια**: Ασφαλιστικά ποσά
- **Άλλο**: Άλλες δαπάνες

### Τύποι Κατανομής

- **Ισόποσα Μερίδια**: Η δαπάνη κατανέμεται ισόποσα
- **Κατά Χιλιοστά**: Η δαπάνη κατανέμεται ανάλογα με τα χιλιοστά
- **Κατά Μετρητές**: Η δαπάνη κατανέμεται ανάλογα με την κατανάλωση

### Επεξεργασία Δαπάνης

1. **Εύρεση**: Βρείτε τη δαπάνη στη λίστα
2. **Επεξεργασία**: Κάντε κλικ στο εικονίδιο επεξεργασίας
3. **Τροποποίηση**: Αλλάξτε τα πεδία που θέλετε
4. **Αποθήκευση**: Αποθηκεύστε τις αλλαγές

### Διαγραφή Δαπάνης

1. **Επιλογή**: Επιλέξτε τη δαπάνη
2. **Διαγραφή**: Κάντε κλικ στο εικονίδιο διαγραφής
3. **Επιβεβαίωση**: Επιβεβαιώστε τη διαγραφή

---

## 💳 Διαχείριση Πληρωμών

### Εγγραφή Νέας Πληρωμής

1. **Επιλογή**: Κάντε κλικ στο κουμπί "Νέα Πληρωμή"
2. **Συμπλήρωση Φόρμας**:
   - **Διαμέρισμα**: Επιλέξτε το διαμέρισμα
   - **Ποσό**: Το ποσό της πληρωμής
   - **Μέθοδος Πληρωμής**: Πώς έγινε η πληρωμή
   - **Ημερομηνία**: Ημερομηνία πληρωμής
   - **Περιγραφή**: Επιπλέον λεπτομέρειες

3. **Αποθήκευση**: Κάντε κλικ στο "Αποθήκευση"

### Μέθοδοι Πληρωμής

- **Μετρητά**: Πληρωμή με μετρητά
- **Τραπεζική Μεταφορά**: Τραπεζική μεταφορά
- **Επιταγή**: Πληρωμή με επιταγή
- **Κάρτα**: Πληρωμή με κάρτα

### Επεξεργασία Πληρωμής

1. **Εύρεση**: Βρείτε την πληρωμή στη λίστα
2. **Επεξεργασία**: Κάντε κλικ στο εικονίδιο επεξεργασίας
3. **Τροποποίηση**: Αλλάξτε τα πεδία που θέλετε
4. **Αποθήκευση**: Αποθηκεύστε τις αλλαγές

## 📊 Μετρήσεις

### Εισαγωγή Μετρήσεων

1. **Επιλογή**: Μεταβείτε στην καρτέλα "Μετρήσεις"
2. **Νέα Μετρήση**: Κάντε κλικ στο "Νέα Μετρήση"
3. **Συμπλήρωση Φόρμας**:
   - **Διαμέρισμα**: Επιλέξτε το διαμέρισμα
   - **Ημερομηνία Μετρήσης**: Ημερομηνία της μετρήσης
   - **Τρέχουσα Τιμή**: Η τρέχουσα ένδειξη του μετρητή
   - **Προηγούμενη Τιμή**: Η προηγούμενη ένδειξη

4. **Αποθήκευση**: Η κατανάλωση υπολογίζεται αυτόματα

### Μαζική Εισαγωγή

1. **Επιλογή**: Κάντε κλικ στο "Μαζική Εισαγωγή"
2. **Ανέβασμα Αρχείου**: Ανεβάστε CSV ή Excel αρχείο
3. **Επιβεβαίωση**: Επιβεβαιώστε τα δεδομένα
4. **Εισαγωγή**: Κάντε κλικ στο "Εισαγωγή"

### Προβολή Στατιστικών

- **Συνολική Κατανάλωση**: Συνολική κατανάλωση κτιρίου
- **Μέση Κατανάλωση**: Μέση κατανάλωση ανά διαμέρισμα
- **Κατανάλωση ανά Διαμέρισμα**: Λεπτομερή στατιστικά

## 🧮 Υπολογισμοί Κοινοχρήστων

### Υπολογιστής Κοινοχρήστων

### Προεπισκόπηση Μεριδίων

Το σύστημα εμφανίζει:
- **Μερίδιο ανά Διαμέρισμα**: Ποσό ανά διαμέρισμα
- **Ποσοστό**: Ποσοστό συμμετοχής
- **Σύνολο**: Συνολικό ποσό

### Έκδοση Κοινοχρήστων

1. **Επιβεβαίωση**: Επιβεβαιώστε τα μερίδια
2. **Έκδοση**: Κάντε κλικ στο "Έκδοση"
3. **Αποθήκευση**: Το κοινοχρήστων αποθηκεύεται
