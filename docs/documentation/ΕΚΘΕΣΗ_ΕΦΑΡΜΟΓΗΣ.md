# 📋 Συνοπτική Έκθεση - Σύστημα Διαχείρισης Πολυκατοικιών

## 🎯 Στόχος και Σκοπός

Η εφαρμογή είναι ένα **ολοκληρωμένο σύστημα διαχείρισης πολυκατοικιών** που επιτρέπει τη διαχείριση πολλαπλών κτιρίων από ένα κεντρικό σύστημα. Σχεδιάστηκε για να εξυπηρετεί διαχειριστές, κατοίκους και διαχειριστικά γραφεία με μια σύγχρονη, φιλική και αποδοτική λύση.

## 🏗️ Αρχιτεκτονική Συστήματος

### Multi-Tenant Architecture
- **Public Schema**: Διαχείριση όλων των tenants (πολυκατοικιών)
- **Tenant Schemas**: Ξεχωριστή βάση δεδομένων ανά πολυκατοικία
- **Django Tenants**: Βιβλιοθήκη για multi-tenancy
- **Docker Containerization**: Εύκολη ανάπτυξη και ανάπτυξη

### Τεχνολογίες
- **Backend**: Django REST Framework (Python)
- **Frontend**: Next.js (React/TypeScript)
- **Database**: PostgreSQL
- **Containerization**: Docker & Docker Compose
- **Styling**: Tailwind CSS
