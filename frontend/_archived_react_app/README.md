# Archived React App

This directory contains the old Create React App (CRA) implementation that was replaced by Next.js.

## Migration Date
October 19, 2025

## What was migrated
- **Old Stack**: Create React App (react-scripts 5.0.1)
- **New Stack**: Next.js 15.5.6 with App Router

## Contents
- `src/` - Old React app source code
- `index.html` - Old React HTML template

## Why archived?
The app was successfully migrated to Next.js for:
- Better performance with Server Components
- Automatic code splitting
- File-based routing
- Better SEO capabilities
- Modern React features

## Do not delete
This is kept for reference in case we need to check the old implementation.
