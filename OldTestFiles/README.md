# OldTestFiles - Test Scripts και Debug Files

Αυτός ο φάκελος περιέχει όλα τα test scripts και debug files που δημιουργήθηκαν κατά την ανάπτυξη του project και μετακινήθηκαν εδώ για καλύτερη οργάνωση.

## Τύποι Αρχείων

### Test Scripts
- **test_*.py**: Βασικά test scripts για διάφορα modules
- **simple_*.py**: Απλοποιημένα test scripts
- **frontend_logic_test.py**: Test για frontend logic

### Debug Scripts
- **debug_*.py**: Scripts για debugging διάφορων issues
- **check_*.py**: Scripts για έλεγχο δεδομένων και καταστάσεων
- **verify_*.py**: Scripts για επαλήθευση δεδομένων

### Fix Scripts
- **fix_*.py**: Scripts για διορθώσεις δεδομένων
- **update_*.py**: Scripts για ενημέρωση δεδομένων
- **add_*.py**: Scripts για προσθήκη δεδομένων

### Utility Scripts
- **run_*.py**: Scripts για εκτέλεση διαφόρων operations
- **final_*.py**: Scripts για τελικούς ελέγχους
- **quick_*.py**: Γρήγορα test scripts
