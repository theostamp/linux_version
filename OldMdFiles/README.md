# OldMdFiles - Αρχεία Documentation

Αυτός ο φάκελος περιέχει όλα τα markdown (.md) αρχεία που υπήρχαν στο project και μετακινήθηκαν εδώ για καλύτερη οργάνωση.

## Περιεχόμενα

- **Project Documentation**: README αρχεία και γενική documentation
- **Implementation Guides**: Οδηγοί υλοποίησης για διάφορα modules
- **Session Management**: Σημειώσεις από development sessions
- **Debug & Fix Documentation**: Documentation για debugging και fixes
- **Completion Summaries**: Συνοψίσεις ολοκλήρωσης features
- **API Documentation**: Documentation για APIs
- **Financial Module Documentation**: Ειδική documentation για το financial module

## Σημείωση

Αυτά τα αρχεία διατηρούνται για ιστορικούς λόγους και αναφορά. Για τρέχουσα documentation, ανατρέξτε στους αντίστοιχους φακέλους του project.
