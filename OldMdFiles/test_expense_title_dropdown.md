# 🧪 Test Guide - Expense Title Dropdown

## 🎯 Στόχος
Test της νέας λειτουργικότητας dropdown επιλογής τίτλου δαπάνης για γρηγορότερη και πιο user-friendly εμπειρία.

## 📋 Test Scenarios

### 1. Βασική Λειτουργικότητα

#### Test Case 1.1: Επιλογή Κατηγορίας
**Βήματα:**
1. Ανοίξτε το modal "Νέα Δαπάνη"
2. Επιλέξτε κατηγορία "ΔΕΗ Κοινοχρήστων"
3. Παρατηρήστε το dropdown τίτλου

**Αναμενόμενο Αποτέλεσμα:**
- Το dropdown τίτλου ενεργοποιείται
- Εμφανίζονται προτάσεις τίτλων
- Εμφανίζεται το μήνυμα "X προτάσεις διαθέσιμες"

#### Test Case 1.2: Επιλογή Τίτλου από Dropdown
**Βήματα:**
1. Επιλέξτε κατηγορία "ΔΕΗ Κοινοχρήστων"
2. Επιλέξτε τίτλο από το dropdown
3. Παρατηρήστε την επιλογή

**Αναμενόμενο Αποτέλεσμα:**
- Ο επιλεγμένος τίτλος εμφανίζεται στο dropdown
- Το custom input δεν εμφανίζεται

### 2. Προσαρμοσμένος Τίτλος

#### Test Case 2.1: Επιλογή Προσαρμοσμένου Τίτλου
**Βήματα:**
1. Επιλέξτε κατηγορία "ΔΕΗ Κοινοχρήστων"
2. Επιλέξτε "✏️ Προσαρμοσμένος τίτλος..."
3. Πληκτρολογήστε προσαρμοσμένο τίτλο

**Αναμενόμενο Αποτέλεσμα:**
- Εμφανίζεται custom input field
- Το input είναι auto-focused
- Μπορείτε να πληκτρολογήσετε προσαρμοσμένο τίτλο

#### Test Case 2.2: Ακύρωση Προσαρμοσμένου Τίτλου
**Βήματα:**
1. Επιλέξτε "✏️ Προσαρμοσμένος τίτλος..."
2. Αφήστε το input κενό
3. Κάντε click εκτός του input

**Αναμενόμενο Αποτέλεσμα:**
- Το custom input κλείνει
- Επιστρέφει στο dropdown
- Δεν αποθηκεύεται κενός τίτλος

### 3. Αλλαγή Κατηγορίας

#### Test Case 3.1: Αλλαγή Κατηγορίας με Επιλεγμένο Τίτλο
**Βήματα:**
1. Επιλέξτε κατηγορία "ΔΕΗ Κοινοχρήστων"
2. Επιλέξτε τίτλο "ΔΕΗ Κοινοχρήστων - Ιανουαρίου 2024"
3. Αλλάξτε κατηγορία σε "ΕΥΔΑΠ Κοινοχρήστων"

**Αναμενόμενο Αποτέλεσμα:**
- Ο τίτλος διατηρείται αν υπάρχει στις νέες προτάσεις
- Αλλιώς εμφανίζεται custom input με τον τίτλο

#### Test Case 3.2: Αλλαγή Κατηγορίας χωρίς Επιλεγμένο Τίτλο
**Βήματα:**
1. Επιλέξτε κατηγορία "ΔΕΗ Κοινοχρήστων"
2. Μην επιλέξετε τίτλο
3. Αλλάξτε κατηγορία σε "ΕΥΔΑΠ Κοινοχρήστων"

**Αναμενόμενο Αποτέλεσμα:**
- Το dropdown επιστρέφει στην αρχική κατάσταση
- Εμφανίζονται οι νέες προτάσεις

### 4. Προμηθευτής

#### Test Case 4.1: Επιλογή Προμηθευτή
**Βήματα:**
1. Επιλέξτε κατηγορία "ΔΕΗ Κοινοχρήστων"
2. Επιλέξτε προμηθευτή "ΔΕΗ ΑΕ"
3. Παρατηρήστε τις προτάσεις τίτλου

**Αναμενόμενο Αποτέλεσμα:**
- Οι προτάσεις τίτλου ενημερώνονται
- Εμφανίζονται προτάσεις με το όνομα του προμηθευτή

### 5. Validation

#### Test Case 5.1: Αποθήκευση χωρίς Τίτλο
**Βήματα:**
1. Συμπληρώστε όλα τα πεδία εκτός από τον τίτλο
2. Παρατηρήστε το validation

**Αναμενόμενο Αποτέλεσμα:**
- Εμφανίζεται error message "Απαιτείται"
- Το dropdown έχει κόκκινο border

#### Test Case 5.2: Αποθήκευση με Προσαρμοσμένο Τίτλο
**Βήματα:**
1. Επιλέξτε "✏️ Προσαρμοσμένος τίτλος..."
2. Πληκτρολογήστε τίτλο "ΔΕΗ Ειδική Χρέωση"
3. Αποθηκεύστε τη δαπάνη

**Αναμενόμενο Αποτέλεσμα:**
- Η δαπάνη αποθηκεύεται επιτυχώς
- Ο προσαρμοσμένος τίτλος διατηρείται

## 🎯 Performance Testing

### Test Case P.1: Ταχύτητα Επιλογής
**Μέτρηση:**
- Χρόνος από επιλογή κατηγορίας έως εμφάνιση προτάσεων: < 100ms
- Χρόνος επιλογής τίτλου από dropdown: < 50ms

### Test Case P.2: Memory Usage
**Μέτρηση:**
- Memory footprint του dropdown component: < 1MB
- Re-renders κατά την αλλαγή κατηγορίας: ≤ 2

## 🐛 Known Issues & Edge Cases

### Edge Case 1: Πολύ Μακροί Τίτλοι
**Περιγραφή:** Τίτλοι με πάνω από 100 χαρακτήρες
**Αναμενόμενο:** Το dropdown πρέπει να εμφανίζει τους τίτλους σωστά

### Edge Case 2: Ειδικοί Χαρακτήρες
**Περιγραφή:** Τίτλοι με ελληνικά, ειδικούς χαρακτήρες
**Αναμενόμενο:** Πρέπει να διατηρούνται σωστά

### Edge Case 3: Πολλές Προτάσεις
**Περιγραφή:** Κατηγορίες με πάνω από 10 προτάσεις
**Αναμενόμενο:** Το dropdown πρέπει να scroll σωστά

## ✅ Success Criteria

### ✅ Functional Requirements
- [ ] Dropdown ενεργοποιείται μόνο μετά την επιλογή κατηγορίας
- [ ] Προτάσεις εμφανίζονται σωστά βάσει κατηγορίας
- [ ] Προσαρμοσμένος τίτλος λειτουργεί σωστά
- [ ] Validation λειτουργεί σωστά
- [ ] Αλλαγή κατηγορίας διαχειρίζεται σωστά

### ✅ UX Requirements
- [ ] UI είναι responsive
- [ ] Loading states εμφανίζονται σωστά
- [ ] Error states διαχειρίζονται σωστά
- [ ] Keyboard navigation λειτουργεί
- [ ] Screen reader compatible

### ✅ Performance Requirements
- [ ] Dropdown ανοίγει < 100ms
- [ ] Επιλογή τίτλου < 50ms
- [ ] Memory usage < 1MB
- [ ] Re-renders ≤ 2 per interaction
