## 🔧 Troubleshooting

### ❌ Κτίριο δεν εμφανίζεται


### ❌ Προηγμένος Υπολογιστής δεν λειτουργεί
```bash
# Ελέγχος API endpoint
curl -X POST http://demo.localhost:8000/api/common-expenses/calculate_advanced/ \
  -H "Content-Type: application/json" \
  -d '{"building_id": 3}'
```

### ❌ Δεδομένα δεν φορτώνονται
```bash
# Επανεκκίνηση container
docker-compose down
docker-compose up -d
```
